# Givifin

Um sistema web simples de gerenciamento de dinheiro com autenticação e perfil.

## Funcionalidades

- **Login Simples**: Autenticação com usuário e senha fixos (admin/admin).
- **Dashboard**: Visualize saldo, adicione receitas e despesas, veja histórico de transações.
- **Perfil**: Configure seu nome e foto de perfil.

## Como Executar

1. Instale as dependências:
   ```
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```
   npm run dev
   ```

3. Abra [http://localhost:3000](http://localhost:3000) no navegador.

4. Faça login com usuário `admin` e senha `admin`.

5. Acesse o dashboard para gerenciar suas finanças e configurar o perfil na aba "Perfil".

## Tecnologias

- Next.js 14
- TypeScript
- Tailwind CSS
- API Routes para backend
